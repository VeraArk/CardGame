Проект «Карточная игра для изучающих иностранный язык». Пользователь может создавать, просматривать и удалять
карточки для изучения иностранных слов.
Каждая карточка будет содержать слово на изучаемом языке и его перевод.


Entity:

Card
Student
Game

Методы

-saveCard;
-deleteCard;
-findCardByTopic;
-findByDifficultyLevel;

-addStudent;
-findAllStudents;
-findStudentByLevel;
-findStudentByUsername;
